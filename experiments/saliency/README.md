# Saliency Map Analysis

## Objective
Analyze which image regions influence the CNN predictions.

## Method
Gradient-based saliency maps were generated for trained models.

## Interpretation
Saliency maps help verify whether the model focuses on relevant spatial regions.

> **Note:**  
> The saliency notebook is provided in a lightweight version without embedded outputs
> to comply with GitHub file size limitations.  
> All qualitative results are available as figures in `experiments/saliency/figures/`.
